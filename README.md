# python tsv to json converter

converts json files to tsv or tsv files to json

can convert json:

    [
        {
            "name": "John",
            "age": 17
        },
        {
            "name": "Micle",
            "age": 24
        }
    ]

to tsv:

    name    age
    John    17
    Micle   24

and vice versa
